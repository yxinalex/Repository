#1 Download
	- https://code.visualstudio.com/

#2 Install
	- VSCode-win32-x64-1.29.1.zip
	- Green version, unzip to execute directly

#3 Package
	- vscode-icons, robertohuertasm.vscode-icons
	- C/C++, ms-vscode.cpptools
	- ctags, hcyang.ctags
	- Vim, vscodevim.vim
	- SVN, johnstoncode.svn-scm
		* Add the path into system environment variable: "C:\Program Files\TortoiseSVN\bin"
	
#4 Configuration
	- Font Size: 12
	- Font Family:
		* Original: Consolas, 'Courier New', monospace
		* ModifyTo: Consolas, 'DejaVu Sans Mono', monospace
		* ModifyTo: 'DejaVu Sans Mono'
